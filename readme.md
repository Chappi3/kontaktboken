# Kontaktboken
Kontaktboken is a console application to manage contact information. Kontaktboken allows you to:
* Create a contact
* Remove a contact
* Print all
* Edit
* Search
* Save to json

## Quick start
    mvn clean package
    java -jar target/kontaktboken*.jar
